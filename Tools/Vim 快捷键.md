## 其他常用快捷键

| 命令   | 动作                     |
| ------ | ------------------------ |
| gg     | 跳至第1行                |
| 7gg    | 跳至第7行                |
| G      | 跳至最后一行             |
|        |                          |
| $      | 跳至行尾                 |
| ^ or 0 | 跳至行首                 |
|        |                          |
| {      | 跳至段首                 |
| }      | 跳至段尾                 |
|        |                          |
| <      | 左缩进                   |
| >      | 右缩进                   |
|        |                          |
| *      | 向后查找当前光标所在单词 |
| #      | 向前查找当前光标所在单词 |
