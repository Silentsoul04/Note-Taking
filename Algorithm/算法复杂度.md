评价算法的好坏：渐近时间复杂度和渐近空间复杂度。

渐近时间复杂度的大O标记：

| 名称         | 运行时间$T(n)$   | 算法举例                     |
| ------------ | ---------------- | ---------------------------- |
| 常数时间     | $O(1)$或者$O(c)$ |                              |
| 对数时间     | $O(log n)$       | 二分查找                     |
| 线性时间     | $O(n)$           | 桶排序                       |
| 线性对数时间 | $O(n*log n)$     | 归并排序、快速排序           |
| 平方时间     | $O(n^2)$         | 选择排序、插入排序、冒泡排序 |
| 立方时间     | $O(n^3)$         | Stupid排序                   |
| 指数时间     | $O(2^n)$         | 汉诺塔                       |
| 阶乘时间     | $O(n!)$          |                              |
